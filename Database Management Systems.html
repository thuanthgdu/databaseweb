Summary:

Môn học giới thiệu các nội dung liên quan đến việc tổ chức, biểu diễn, cập nhật và khai thác thông tin trên một hệ quản trị cơ sở dữ liệu quan hệ - SQL server. Nội dung chính gồm cách thức quản trị (tạo mới, cập nhật, tìm kiếm và truy vấn, xóa) cơ sở dữ liệu và các thành phần của nó (bảng dữ liệu, bảng ảo, bảng chỉ mục, thủ tục thường trú, v.v.) bằng các câu lệnh SQL và bằng công cụ trực quan SQL Server Management Studio.

Contents & Lectures: (các file sql đi kèm bài giảng down tại đây)

  • Lec 1: Giới thiệu (PPT)
  • Lec 2: T-SQL for Data Definition (PPT)
  • Lec 3: T-SQL for Data Definition & Data Manipulation (Tiếng Việt, PPT)
  • Lec 4: Indexes (PPT), Data Integrity (PPT)
  • Lec 5: Lập trình T-SQL (PPT)
  • Lec 6: View, Stored Procedure, User Definited Function, Trigger (PPT)
  • Lec 7: T-SQL for Data Control  (PPT)
  • Lec 8: Transaction and Lock

Assignments:

1. Cài đặt phần mềm SQL Server và SQL Server Management Studio (Download từ đây)

2. Sử dụng SQL Server Management Studio và sqlcmd (start --> run --> cmd --> sqlcmd) để connect với SQL Server

3. Cài đặt 2 cơ sở dữ liệu mẫu (có dữ liệu): pubs và northwind vào SQL server (download tại đây). Cài bằng 2 cách: (1) thực hiện các lệnh T-SQL trong các file instpubs.sql và instnwnd.sql để sinh ra các databases, (2) attach các databases đã có sẵn (PUBS.MDF, PUBS.LDF; NORTHWND.MDF, NORTHWND.LDF).

4 . Tìm hiểu cấu trúc các tables, views, stored procedures trong các cơ sở dữ liệu pubs và northwind. Tạo các data diagram từ tất cả các bảng trong các cơ sở dữ liệu này.

5. Bài tập thực hành số 1 (down tại đây)
